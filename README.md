# Components
- 25mm DC 12V GA-370 Low Speed Metal Gear Motor 400RPM 3.5 Watts 90 g
- TB6612FNG Dual Driver Controller Board MOSFET H Bridge 15V max VCC=2.7-5.5V Output 1.2A/3.2A
- 12.6V 3000mAh Lithium Ion Battery 157 g
- OT253-B47 Step-down buck converter 4.5-24V to 5V 3A
- HC-SR04 Ultrasonic Sensor
- TCRT5000 Reflective Infrared Optical Sensor
- TSSP4038, 38kHz IR Receiver, 950nm ±45°, 25m Range, 2.5-5.5V
- HC4067 16-channel Analog Multiplexer
- BMM150 Triple Axis Magnetometer Sensor
- MCU: ESP32-S3
